Installation


Play CDN

First, import Transfer in your JavaScript file.


script.js

import Transfer from "https://cdn.transferdataweb.repl.co/v1.js"

Then, create a variable named "transfer", that corresponds to the Transfer Library.


script.js

import Transfer from "https://cdn.transferdataweb.repl.co/v1.js"
let transfer = new Transfer();


Manually

First, go to https://cdn.transferdataweb.repl.co

Then, download the file or copy everything

Finally, upload the downloaded file or paste the copied content to your script.js file.
