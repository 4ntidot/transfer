How to use :cross?

:cross is the data you will send to other pages.

You can put any kind of value in :cross. A string, a list, a number... every kinds are accepted.


How to use :target?

If you want your data to be sent to a specific page, put the path of this page in the :target value.

Warning: You can't put several pages in :target. If you want to send it to several pages,you don't have any kther choice than leaving :target null.
